How it works:

Generate

Simply click on the "Generate" button above and a new palette of colors and a frontpage image will be generated. The newly generated colors will update on this page to help you see if the design is right for you. You can enter an optional starting hex value above the Generate button. The palette will be built around the color you enter.

Lock

You can lock a color by clicking on that color's block at the top of the page. Locked colors won't change (much) when generating a new design.

Save

Once you've decided you like a particular design, press the save button. Your design will show up in the "Saved Palettes" section.

Revisit

Try as many designs as you want! There's no limit to the amount you can save.

Why do you use a proxy server?

The simple explanation:

It helps us retrieve our color palette data by acting as a middleman between where we get our data and where we host our web page.

The more technical explanation:

We're hosting our website using GitHub Pages, which uses the HTTPS protocol. The ColorMind API, where we receive our color palettes, uses the HTTP protocol (notice the lack of "S"). As a result of the difference between these two protocols, most modern browsers block the ColorMind API. CORS Anywhere, our proxy server, acts as an intermediary and sends ColorMind's data to GitHub Pages using the HTTPS protocol. This allows your browser to accept palettes coming from the ColorMind API.
